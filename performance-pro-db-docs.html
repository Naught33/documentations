Database API Documentation

This documentation covers the usage of all database functions in db.ts which interact with a Supabase PostgreSQL database.

Note: Before using any functions, ensure you have these environment variables configured:
SUPABASE_PROJECT_ID="your-supabase-project-url"
SUPABASE_API_KEY="your-supabase-anon-key"

General Patterns

All database functions follow these patterns:

  • Insert functions: Take a complete object of the specified type and create a new record
  • Update functions: Take an ID, field name, and new value to update a specific field
  • Delete functions: Take an ID to delete a specific record
  • Get functions:
    • getX(id: string) - Get a single record by ID
    • getXs(startPage?, endPage?) - Get multiple records with optional pagination

User Operations

insertUser(user: User): Promise<void>

Creates a new user.

Example:
const newUser: User = {
  _id: 'user123',
  email: 'test@example.com',
  role: 'admin',
  supabaseid: 'supa123',
  subscriptionplane: 'premium',
  createdat: new Date()
};
await insertUser(newUser);
updateUser(id: string, field: string, value: any): Promise<any>

Updates a specific field of a user.

Example:
await updateUser('user123', 'email', 'new@example.com');
deleteUser(userId: string): Promise<void>

Deletes a user by ID.

getUser(userId: string): Promise<any>

Gets a single user by ID.

getUsers(startPage?: number, endPage?: number): Promise<any>

Gets all users with optional pagination.

Examples:
// Get all users
const allUsers = await getUsers();

// Get users 10-20
const pagedUsers = await getUsers(10, 20);

Ad Operations

insertAd(ad: Ad): Promise<any>

Creates a new ad.

Example:
const newAd: Ad = {
  _id: 'ad123',
  userid: 'user123',
  title: 'Summer Sale',
  platform: 'facebook',
  campaignstart: new Date('2023-06-01'),
  campaignend: new Date('2023-06-30'),
  uploadedat: new Date()
};
await insertAd(newAd);

Error Handling

All functions throw errors when:

  • Required parameters are missing
  • Database operations fail
  • Invalid operations are attempted (like updating an _id field)

Always wrap database calls in try/catch blocks:

try {
  await insertUser(newUser);
} catch (error) {
  console.error('Database error:', error);
}
